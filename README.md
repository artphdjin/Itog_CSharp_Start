## Итоговая работа студента группы 2334 Панкратьев Артем Игоревич по курсу "Знакомство с языками программирования (C#)"

### Задача Итоговой работы:
Написать программу, которая из имеющегося массива строк формирует другой массив из строк, длина которых <= 3 символа.
Первоначальный массив можно ввести с клавиатуры (наш вариант), либо задать на старте выполнения алгоритма.
При решении алгоритма не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.

### Итоговая работа включает:
1. Создать репозиторий на гитхаб.
2. Составить блок-схему в io draw - подгрузить в репозиторий
3. Написать README.md - подгрузить в репозиторий
4. Написать программу, решающую **Задачу**
5. Использовать контроль версий (раздельные коммиты для п.2, 3, 4)

## Программа
### Этап 1: Ввод данных в массив и проверка соответствия **Условию**
1) Объявляем консту limitLength предельной длины строки, допустимой для переноса в массив вывода.
2) Дублирует в комментарии текст Задачи.
3) Используя цикл проверки корректности введённого типа данных - вводим размерность массива incomeArray_limit.
4) Объявляем строковый массив incomeArray[] для вводимых строк, а также массив bool OutcomeArray_Validation[] для хранения идентификатора удовлетворения **Условию**
.Length <= limitLength, для переноса в массив вывода.
5) Объявляем int OutcomeArray_limit - размерность массива вывода.
6) в цикле с количеством шагов incomeArray_limit построчно вводим строки в incomeArray[],
7) затем в том же цикле, добавляем в OutcomeArray_Validation[] запись соответствия **Условию** и увеличиваем OutcomeArray_limit в случае соответствия.

### Этап 2: Создание массива вывода и Вывод этого массива на экран.
1) Проверка того, что OutcomeArray_limit > 0 (необходимые строки найдены), иначе выводим сообщение, что такие строки не найдены и не создаём массив.
2) После успешной проверки объявляем массив вывода string[] OutcomeArray с размерностью OutcomeArray_limit ,
3) а также объявляем int idx контроллируемый счётчик порядка вводимого элемента
4) для каждого элемента одновременно incomeArray[] (каждому из которых соответствует OutcomeArray_Validation[]) проверяем значение OutcomeArray_Validation[],
если удовлетворяет **Условию**, то мы заносим данный элемент incomeArray[] в OutcomeArray[], увеличиваем счётчик idx,
а так же, проверяя является ли этот элемент 1м, последним или между ними, различным образом выводим элементы нового массива на экран.
5) Делаем отступ для удобства чтения вывода.
